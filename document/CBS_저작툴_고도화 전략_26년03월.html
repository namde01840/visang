CBS 저작툴 현황 › 동적 콘텐츠 저작툴

트리거-액션 기반 조작형 콘텐츠 저작

복잡한 코딩 없이 트리거-액션 기반(예: 버튼 클릭 시 모션 실행)의
직관적인 로직으로 복잡도 높은 조작형 콘텐츠를 생산하기 위해 구축된 저작툴임

1차 구축 완료 25년 11월

  • 오브젝트 생성 및 배치
  • 애니메이션 및 모션 적용
  • 미리보기 뷰어 및 내보내기

구축 배경: 조작형 콘텐츠의 외주 의존도가 높고,
사후 유지보수에 많은 공수가 드는 문제점을 해결하기 위해 구축됨

현재 서비스의 Pain Point 분석

1) 범용 저작툴

✓ 강점: 교과급 및 과목별 광범위한 템플릿의 콘텐츠를 빠르게 양산하는 데 최적화됨

✗ Pain: 너무 많은 기능을 수행하다 보니 입문자가 플랫폼에 온보딩하여 전체 기능을 파악하는 데 높은 학습 비용이 발생함

2) 동적 콘텐츠 저작툴

✓ 강점: 별도의 코딩 없이 콘텐츠 저작자가 트리거-액션 기반의 조작형 콘텐츠를 생산하고 편집하는 데 최적화됨

✗ Pain: 객체 및 모션을 생성하는 UI가 입문자에게 다소 어렵게 느껴질 수 있음

3) AX 트렌드 대응 필요

AX 트렌드에 맞춰 손쉽게 콘텐츠를 제작하고 기존 콘텐츠를 재활용할 수 있는 방안을 마련하여 경쟁력을 제고해야 함

Insight

'빈 화면'에서 시작하는 기존 방식에서 탈피하여, 기 구축된 방대한 콘텐츠를 손쉽게 재활용하고 즉각적으로 재생산할 수 있는 지능형 방안 도입이 필수

고도화 전략 › Step 1. 설정값 기반 초벌 콘텐츠 생성

저작 시작 화면 개편

저작툴 첫 화면에서 CBS 메타정보 옵션값을 선택하고 프롬프트를 직접 입력할 수 있도록 함

  • 템플릿 유형 / 난이도 / 교과 / 학년 / 학교급

AI 추론 기반 콘텐츠 추천

CBS 벡터 DB를 기반으로 AI가 추론하여
가장 정확한 콘텐츠(혹은 템플릿)을 선제적으로 추천하는 방식을 도입함

기대 효과

1) '빈 화면에 대한 공포'를 해소하고, 입문자의 진입 장벽을 완화함
2) 정교하게 선별된 DB를 제공하여 교육자료의 신뢰성을 확보하고, 익숙한 레이아웃에서 편집을 시작할 수 있음

고도화 전략 › Step 2. 대화 기반 편집 옵션 제공

대화 기반 편집

화면 이동 없이 초벌 콘텐츠에 대한 편집을 기존 저작 플로우 내에서 이어서 할 수 있도록 AI Assistance 기능을 제공함

  • 동적 저작툴: 객체 및 액션 생성 메뉴에 AI 챗봇 제공
  • 범용 저작툴: AI가 키워드 형식으로 커스터마이징 제안

AI 자동 편집

레이아웃, 오탈자 등을 자동 검수하여 콘텐츠 품질을 보장함

고급 편집 기능

기존 저작 방식을 그대로 제공하여 수동으로 정교한 편집이 가능하도록 함

기대 효과

다양한 편집 옵션(대화형 / AI 자동 / 고급 수동)을 제공하여 저작자의 숙련도에 관계없이 최적의 편집 경험을 보장함
