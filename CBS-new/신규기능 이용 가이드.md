# 신규기능 이용 가이드

> 레이어 팝업 형태로 제공되며, 3개 탭(AI 지문 생성 / AI 난이도 변경 / AI 문제 생성)으로 구성
> 탭 전환 시 스크롤이 자동으로 상단 초기화됨

## AI 지문 생성

1. 지문 관리 목록 상단 **AI 생성** 버튼 클릭 → **AI 지문 생성** 선택
   - 참고 이미지: 이미지 (2).png

2. **생성** 버튼 클릭 → 학년 / 난이도 / 단어 수 / 글 형식 / 소재·주제 등 생성 조건 설정 → 설정 완료 후 **생성하기** 버튼 클릭
   - 참고 이미지: 이미지 (3).png

3. 지문 생성 완료 후 **CBS 저장** 버튼 클릭
   - 참고 이미지: 이미지 (4).png

4. 생성된 지문 확인 후 교과 / 교과과정 / 타이틀 등 정보 입력 → **내보내기** 버튼 클릭
   - 참고 이미지: 이미지 (5).png

5. 지문 관리 목록에서 저장된 지문 확인 또는 수정 메타 정보 수정
   - 참고 이미지: 이미지 (7).png

---

## AI 난이도 변경

1. 지문 관리 목록 상단 **AI 생성** 버튼 클릭 → **AI 난이도 변경** 선택
   - 참고 이미지: 이미지 (2).png

2. 난이도를 변경할 지문 선택
   - 참고 이미지: 이미지 (8).png

3. 원하는 난이도 선택 (**쉽게 / 비슷하게 / 어렵게**) 후 저장 클릭
   - 참고 이미지: 이미지 (9).png

4. 변환된 지문 확인 후 교과 / 교과과정 / 타이틀 등 정보 수정 수 저장
   - 참고 이미지: 이미지 (10).png

---

## AI 문제 생성

1. 지문 화면에서 **AI 문제 생성** 버튼 클릭 후 생성할 문제 유형 선택 (최대 5개 선택 가능) → **PDF 다운받기** 또는 **생성** 버튼 클릭
   - 참고 이미지: 이미지 (14).png

2. **생성** 버튼 클릭 시 AI가 생성한 문제를 저작툴에서 확인 및 수정
   - 참고 이미지: 이미지 (15).png

3. 각 콘텐츠별 정보 입력 후 저장
   - 참고 이미지: 이미지 (16).png

4. 저장된 문제는 **일반 콘텐츠 관리**에서 아티클 형태로 확인 및 활용 가능
   - 참고 이미지: 이미지 (17).png
